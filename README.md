# 📋 Журнал посещаемости

Веб-приложение для учета посещаемости студентов с поддержкой:
- 5 дней недели
- 4 пар в день
- Drag-and-drop интерфейсом
- Автосохранением в localStorage
- Адаптивным дизайном

## Функционал
- Отметка присутствия/отсутствия студентов
- Drag-and-drop перемещение между списками
- Кастомизация названий пар и дат
- Сводка по каждому занятию
- Автосохранение всех изменений

## Технологии
- Чистый HTML/CSS/JS
- LocalStorage для хранения данных
- Drag-and-drop API
- Полностью адаптивный дизайн
